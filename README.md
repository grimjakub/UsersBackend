## Application
### Installation
install python requirements
```
pip install -r requirements.txt
```
### Usage
For simple use
```
http://127.0.0.1:5000/[route]
```
For example get all users
```
http://127.0.0.1:5000/users
```
For example get user 'Bob'
```
http://127.0.0.1:5000/user/Bob
```
